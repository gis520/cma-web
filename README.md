# cma-web

气象数据综合展示平台，使用 webpack 环境开发的，基于 `leaflet.js`

## WebGIS 开发教程

- [WebGIS 入门实战](https://xiaozhuanlan.com/webgis)

## 使用方式

根目录执行: `npm install` ，安装依赖

本地启动： `npm run start`，打开 `http://localhost:8080/`

打包部署： `npm run build` ，将生成的 `dist` 目录下的文件部署即可。
